# Changelog

## Unreleased

### Added
- `PreviewShip: Deploy Active HTML File` 命令 — 将当前打开的 `.html` 文件作为 `index.html` 打包并部署
- `PreviewShip: Deploy Workspace or HTML File` 命令 — 在当前编辑器为 HTML 文件时，可选择部署单个 HTML 或整个工作区

## [0.1.0] - 2026-03-17

### Added
- `PreviewShip: Set API Key` 命令 — 安全存储 API Key
- `PreviewShip: Deploy Current Workspace` 命令 — 一键打包上传部署
- `PreviewShip: Show Usage` 命令 — 查看剩余部署额度
- 状态栏实时显示部署进度
- 可配置排除规则、服务器地址、轮询参数
- 完整的错误提示和交互操作
