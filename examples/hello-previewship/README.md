# Hello PreviewShip

A minimal browser-ready static site for testing PreviewShip without a framework or build step.

## Publish it

Choose either route:

- Open [PreviewShip Try](https://previewship.com/try), zip this folder, and upload it.
- Run `npx previewship deploy ./examples/hello-previewship` after configuring a PreviewShip API key.

The folder contains `index.html`, so it can be served as-is. Repeated deployments to the same project keep the same project URL.
